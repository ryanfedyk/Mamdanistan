# Welcome to Mamdanistan — Project Rules

## Tech Stack
- Frontend: Next.js / Tailwind CSS
- Mapping: Mapbox GL JS (or Leaflet)
- State: Local browser localStorage for arcade data / user progress

## Development Constraints
- Keep it highly optimized for mobile web (target low-end phones for viral access).
- Use modular components for different "zones" or features.
- Test commands: `npm run lint` and `npm run build`.
